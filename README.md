# Swarm Self-Evolving — 多 Agent 自我进化系统

9 个 Hermes Agent 分成两队，自动写代码、建 Skill、积累记忆、提交 Git。零人工干预。

## 架构

```
协调者 Agent 0 ─── 调度/决策/Git提交
     │
  ┌──┴──┐
  A队   B队
  开发   审查
4 Agent 4 Agent
```

## 开发阶段（每天15-30分钟）

| 天 | 任务 | 状态 |
|----|------|------|
| D1 | 项目初始化 | ✅ |
| D2 | 本地模型部署 (Ollama) | ⬜ |
| D3 | Git 仓库配置 | ⬜ |
| D4 | 单 Agent 试跑 | ⬜ |
| D5 | 创建 cronjob | ⬜ |
| D6 | 第一轮监控 | ⬜ |
| D7+ | 自动运行 | ⬜ |

详细规划见 `开发工单.md`。
